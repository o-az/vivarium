## Coding Conventions

- Use kebab-case for filenames: server, components, hooks, all
- Seldom use abbreviations. For example:
  - instead of `e`, use `event`,
  - instead of `c`, use `context`,
  - instead of `cb`, use `callback`
- Prefer `Array<>` over `[]`
